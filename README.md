# WanderLust Tourism Company Website

A modern, responsive tourism company website built with HTML, CSS, and JavaScript.

## Features

### Homepage (`index.html`)
- **Hero Section**: Stunning image slider with automatic transitions
- **Navigation**: Fixed responsive navigation with smooth scrolling
- **Search Functionality**: Interactive destination search with date picker
- **Destinations**: Popular destinations showcase with hover effects
- **Packages**: Travel packages with detailed information and booking
- **About Section**: Company information with animated statistics
- **Testimonials**: Customer reviews with auto-sliding carousel
- **Contact Form**: Functional contact form with validation
- **Newsletter**: Email subscription functionality

### Destination Pages (`paris.html`)
- **Destination Hero**: Parallax background with destination information
- **Quick Info**: Essential travel information (weather, currency, language)
- **Photo Gallery**: Interactive image gallery with thumbnail navigation
- **Top Attractions**: Detailed attraction cards with ratings and pricing
- **Travel Packages**: Destination-specific packages
- **Travel Tips**: Helpful advice for travelers
- **Enhanced Booking**: Comprehensive booking form with price calculation

## Setup Instructions

1. Download all files to your web server directory
2. Ensure all files are in the same folder
3. Open `index.html` in a web browser
4. For full functionality, serve from a web server (not just file://)
